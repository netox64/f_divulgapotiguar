## Prerequisites

- Nodejs v 22
- create archive .env in folder root of project

```
    NEXTAUTH_URL = http://localhost:3000
    NEXTAUTH_SECRET = secret
```

## Running_Application
- run
```
    npm i && npm run dev
```
- run storybook
```
    npm i && npm run storybook
```
